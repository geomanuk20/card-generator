# Card Generator

A professional MERN stack application for creating high-quality news cards with AI background removal and multi-subject support.

## Features
- **AI Background Removal:** Manual trigger for clean subject cutouts.
- **Dual Subjects:** Support for main and sub-subject images with independent styling.
- **Granular Controls:** Precise positioning, sizing, and focus (corners/centers).
- **Premium Typography:** Integrated Google Fonts with full styling controls.
- **Live Preview:** Real-time visual feedback while editing.
- **High-Quality Export:** Download cards as PNG images.

## Tech Stack
- **Frontend:** React, Vite, Axios, html-to-image
- **Backend:** Node.js, Express, Mongoose, Multer
- **AI:** @imgly/background-removal-node

## Setup
1. Clone the repository.
2. Run `npm install` in both `frontend` and `backend` directories.
3. Add your MongoDB connection string to `backend/.env`.
4. Run `npm run dev` in both directories to start the application.
